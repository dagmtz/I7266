\subsection*{Actividades}
\begin{enumerate}[label=\arabic*)]
    \item Escribe y compila un programa en C que pida un número entero positivo y después imprima la suma de todos los número naturales hasta ese número.
    \item Escribe y compila un programa en C que pida un número entero positivo y después imprima la tabla de multiplicar del 1 al 12 del número ingresado.
    \item Escribe y compila un programa en C que pida un número entero positivo y después imprima el factorial de ese número.
\end{enumerate}
En todos los casos hay que subir el archivo fuente, además de un reporte que incluya los detalles de la compilación y el funcionamiento de los programas. Código generado por IA no será aceptado si el alumno no puede justificar su funcionamiento. Para compilar los programas se debe usar GCC, se sugiere usar el siguiente flujo de trabajo para cada programa:
\begin{enumerate}
    \item Crear una nueva carpeta (sugerencia extra: numerada), por ejemplo: \texttt{001\textunderscore suma}
    \item Dentro de la carpeta, crear un archivo de texto que se llame \texttt{main.c}
    \item Copiar la dirección de la carpeta, esta se puede obtener desde la barra de dirección del explorador de archivos
    \item Abrir una nueva instancia de Powershell (desde el menú de Windows)
    \item Utilizar el comando: \texttt{cd ``C:\textbackslash direccion\textbackslash de\textbackslash la\textbackslash carpeta''} (substituyendo entre las comillas por la dirección que se copió)
    \item Dentro de la carpeta, usar el comando \texttt{gcc -Wall main.c -o main.exe} para compilar y generar un ejecutable
    \item Para ejecutar el programa, usar el comando \texttt{.\textbackslash main.exe}
\end{enumerate}
