\subsection*{Actividades}
Investiga y escribe una breve síntesis sobre lo que hayas encontrado de cada uno de los siguientes elementos de una CPU:
\begin{itemize}
    \item Arithmetic Logic Unit
    \item Control Unit
    \item Registers:
    \begin{itemize}
        \item General Purpose Register
        \item Special Function Registers:
        \begin{itemize}
            \item Accumulator
            \item Program Counter
            \item Stack Pointer
            \item Instruction Register
            \item Status Flags/Registers
        \end{itemize}
        \item Instruction Decoder
        \item Buses
        \begin{itemize}
            \item Data bus 
            \item Address bus 
            \item Control bus     
        \end{itemize}
        \item Clock signals
        \item Interrupt Controller
    \end{itemize}
\end{itemize}
Con base en la información que escribas, responde las siguientes preguntas:
\begin{enumerate}
    \item ¿Cómo se relacionan la ALU y la Control Unit?
    \item ¿A qué parte de la memoria de un microcontrolador apunta el Program Counter?
    \item ¿Y a qué parte apunta el Stack Pointer?
    \item ¿Por qué se separan los registros de propósito general y específico?
    \item ¿De dónde proviene la señal de reloj?
\end{enumerate}
